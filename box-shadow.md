# box-shadow

## 參數設定

```
box-shadow: X偏移px Y偏移px 模糊的程度px 外擴或内縮px 指定顏色rgba
```

## 範例

```pug
div 這是陰影框
```

```sass
div
	width: 350px
	height: 200px
	background-color: #243010
	color: white
	text-align: center
	line-height: 200px
	box-shadow: 10px 10px 20px -5px rgba(0,51,0,0.5)
```
