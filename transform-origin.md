# transform-origin

```
transform-origin: X軸瞄點 Y軸瞄點
```

## X軸瞄點，可填入

* `center` (=`50%`)
* `left` (=`0`)
* `right` (=`100%`)
* `○○px`
* `○○%`

## Y軸瞄點，可填入

* `center` (=`50%`)
* `top` (=`0`)
* `bottom` (=`100%`)
* `○○px`
* `○○%`
