# 對話框效果

## 做法剖析

用`background-color`做普通的方框+用複雜的`border`做三角形

```pug
h1 這是一段標題
```

```sass
h1
 background: teal
 color: #fff
 padding: 10px 20px
 position: relative
 &:after
  content: ''
  border-right: 15px solid transparent
  border-left: 15px solid transparent
  border-top: 25px solid teal
  position: absolute
  bottom: -25px
  left: 30px
```
